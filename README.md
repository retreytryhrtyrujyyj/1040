# 超市积分管理系统

## 项目简介
超市积分管理系统是基于Java语言，Servlet、JPS技术、Mysql数据库的超市积分管理系统。两类用户：管理员和普通用户。模块包括：用户管理、商品管理、积分管理等。适合毕业设计、大作业、课程设计。

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=1040

## 技术介绍
Java;JPS;Mysql

## 开发工具
1.IntelliJ IDEA或者Myeclipse 3.Navicat 3.Mysql 4.JDK

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
